# GRPC

## 什么是 API

在其核心，应用程序编程接口（API）是作为两个单独的软件之间的保证合同的正式规范。 API提供程序定义了它期望的一组操作，数据格式和协议。API的消费者（称为客户端）将使用那些规则，客户端将永远不必担心API本身的内部，只要它遵循规则。

## API 技术

* TCP / IP 套接字
* 远程过程调用（RPC）
* 公共对象请求代理体系结构（CORBA）
* Java 远程方法调用（RMI）和企业Java Bean（EJB）
* Microsoft 分布式组件对象模型（DCOM） - 也称为ActiveX
* Web 服务（SOAP 然后 REST）
